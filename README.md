## Netview
Ce projet est un clone de Netflix avec la plupart des fonctionnalités à savoir: 
* L'inscription
* La connexion
* Continuer la série ou le film au dernier moment
* La recherche 
* Proposition de vidéos que vous pourrez aimer 
* ...
